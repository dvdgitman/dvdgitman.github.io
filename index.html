Featured Project

Automated Home Media Server

Docker Automation Homelab Linux

To apply DevOps principles in a practical, hands-on environment, I built a fully automated personal media server. The entire system, including Plex, Sonarr, and Radarr, is defined and managed in a single `docker-compose.yml` file, making it a true Infrastructure as Code project.

version: "3.7"
services:
  plex:
    image: lscr.io/linuxserver/plex:latest
    container_name: plex
    network_mode: host
    environment:
      - PUID=1000
      - PGID=1000
      - TZ=Asia/Jerusalem
    volumes:
      - /path/to/plex/config:/config
      - /path/to/media/tv:/tv
    restart: unless-stopped
