Featured Project

Dockerized Homelab Infrastructure

Docker Compose IaC Cloudflare Tunnel Uptime Kuma Plex

This repository is the single source of truth for my personal server infrastructure, managed entirely with Docker Compose. It's a real-world example of applying Infrastructure as Code (IaC) principles to a homelab environment, allowing for rapid deployment, easy recovery, and version-controlled configuration of all services.

# --- cloudflared.yaml ---
# Securely exposes web services to the internet
version: "3.6"
services:
  cloudflared:
    container_name: cloudflared
    image: cloudflare/cloudflared:latest
    restart: unless-stopped
    command: tunnel --no-autoupdate run --token [YOUR_TOKEN]

# --- uptime-kuma.yaml ---
# Self-hosted monitoring for all my services
version: '3.3'
services:
  uptime-kuma:
    image: louislam/uptime-kuma:1
    container_name: uptime-kuma
    volumes:
      - ./data:/app/data
    ports:
      - 3001:3001
    restart: unless-stopped

# --- plex.yaml ---
# Media server for streaming content
version: "3.7"
services:
  plex:
    image: lscr.io/linuxserver/plex:latest
    container_name: plex
    network_mode: host
    environment:
      - PUID=1000
      - PGID=1000
    volumes:
      - ./config:/config
      - /share/media:/media
    restart: unless-stopped
